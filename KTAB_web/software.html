KAPSARC Toolkit for Behavioral Analysis

A Framework for Modeling Collective Decision-Making Processes (CDMPs)

The initial model that has been developed within the KTAB framework, with an accompanying user interface (UI), is the Spatial Model of Politics (SMP). Other models are available in the source code.

The KTAB SMP is available for download on Windows, Linux, and macOS. The code is compiled, linked, and tested with absolutely no modification, simultaneously on both Windows and Linux: there is one single body of cross-platform code, not separate versions for each OS, which allowed it to be cross over to macOS. It was developed and tested on 64-bit Windows and Linux platforms, including the following:

KTAB is currently developed as libraries of code to enable analysts and computer programmers to construct their own models of CDMPs, secure in the knowledge that their approach is theoretically sound. These are already available for free download at https://github.com/KAPSARC/KTAB

Installing KTAB SMP on Windows

Installation of the SMP application on Windows is simple.
1. Download and install, Microsoft Visual C++ Redistributable for Visual Studio 2015 or higher from here
2. Click the Download for Windows button at the top of this page to download a zip archive of the application compiled for 64-bit machines, along with all the required shared libraries.
3. Extract the contents of the archive to your PC, maintaining the directory structure in the archive
Nothing else should be required to run the application executable KTAB_SMP.exe.
